# CQB Simulator — 室内近距离战斗模拟器

**by KiraFaye  |  Ver 2.0**

> 纯前端战术标图与 CQB 战术规划工具，打开浏览器即可使用。

## 技术架构

- **前端语言**：HTML5 + CSS3 + Vanilla JavaScript（ES6 Class）
- **渲染方式**：Canvas 2D API 自绘一切元素，零第三方依赖
- **状态管理**：楼层 → 场景 → 元素三级树结构，配合 JSON 快照历史栈实现撤销/重做（50步）
- **UI 主题**：SpaceX 航空航天美学（纯黑背景 + 光谱白文字 + 幽灵边框）
- **响应式**：桌面端 / 移动端双端适配，媒体查询 + 触控手势

***

## 核心功能

### 1. 多楼层多场景系统

- 支持多楼层（1F、2F……），可增删移动重命名
- **每个楼层内支持多个独立场景**，场景间完全隔离存储元素与枪线
- 新建场景自动复制前一场景的全部内容
- 场景管理：双击重命名（桌面）/ 长按弹窗操作（移动端）、hover×删除
- 楼梯可绑定目标楼层实现跨层连接

### 2. 建筑元素绘制

| 元素     | 功能                                    |
| ------ | ------------------------------------- |
| 🧱 墙体  | 矩形，任意旋转，遮挡视野                          |
| 🚪 门   | 4向铰链（左/右/上/下）+ 开合角度 0°\~360°，颜色跟随元素属性 |
| 🪟 窗户  | 半透明，不阻挡视野穿透                           |
| 🪜 楼梯  | 跨楼层通道                                 |
| 🚧 遮挡物 | 矩形或圆形，支持旋转                            |

所有元素均支持拖拽、旋转、缩放、改色、标注标签

### 3. 人员单位

- 👤 操作员（友方）— 名称/呼号/职位，可作为枪线起点
- 💀 敌人 — 实战模式下可隐藏，可作为枪线起点
- 🙋 人质 — 不可作为枪线起点
- 枪线绘制：桌面端右键长按拖动 / 移动端单指长按350ms拖动

### 4. 视野遮挡系统（核心特色）

- 扇形 FOV（默认120°），距离可设有限/无限
- **Raycasting 射线投射算法**：72条射线扫描所有障碍物
- 旋转建筑元素按每个角度实时计算遮挡多边形
- 门的开合状态动态影响视野穿透（4向铰链完整支持）
- 可全局开关遮挡显示

### 5. 画板工具

- ✏️ 自由画笔 / ▭ 矩形 / ⭕ 圆形 / 📏 直线 / 🗑️ 区域擦除 / 🧹 清空
- 颜色自由选取，线宽 1\~20px

### 6. 画布缩放与平移

| 操作 | 桌面端          | 移动端  |
| -- | ------------ | ---- |
| 缩放 | 鼠标滚轮（以光标为中心） | 双指捏合 |
| 平移 | 选择模式+空白处拖拽   | 单指拖动 |

范围：25% \~ 500%，坐标自动转换

### 7. 智能对齐

- 网格吸附 + 元素边缘吸附（阈值8px）
- **尺寸吸附**：创建时自动匹配已有元素宽高
- 快捷键 S 切换开关

### 8. 其他功能

- ⚔️ 实战模式（隐藏敌方，快捷键 M）
- ↩️↪️ 撤销/重做（Ctrl+Z/Y，移动端底部圆形按钮）
- 📷 导出 PNG（Ctrl+S）
- 🔲 网格切换（G）

## 数据架构

```
Layers[]
  └─ Layer { id, name, scenes[], currentSceneId }
       └─ Scene { id, name, elements[], gunlines[] }
            └─ Element { type, x, y, width, height, rotation, color... }
```

撤销/重做快照保存完整的 `scenes[]` 结构，确保跨场景操作可回溯。

## UI 设计

### 视觉风格

- **纯黑背景** `#000000` + **光谱白文字** `#f0f0fa`
- 半透明幽灵边框/表面（Ghost Surface）
- 大写字母铭牌风格（字间距）
- 药丸形激活态（白底黑字 + 光晕）

### 楼层管理区布局

```
┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
│ 1F 一楼        ▲ ▼ ×    │ ← 名称(粗体) + 按钮(hover显)
│ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ │   细分隔线
│ [场景1] [+]           │ ← 药丸Tab, 激活=反色高亮
└─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘
```

- 操作按钮默认隐藏，hover/active 时淡入
- 场景Tab：点击切换、双击编辑、hover×删除、移动端长按弹面板
