================================================================================
                         CQB Simulator - 室内近距离战斗模拟器
                              by KiraFaye  Ver2.0
================================================================================

【项目概述】

CQB Simulator（室内近距离战斗模拟器）是一款基于纯前端技术构建的
战术标图与 CQB 战术规划工具。无需安装任何依赖，打开浏览器即可使用。
适用于战术规划、室内作战推演、CQB 训练教学等场景。

技术栈：HTML5 Canvas + 原生 JavaScript + CSS3
文件结构：index.html / app.js / spacex-ui.css


================================================================================
                                 核心功能
================================================================================

一、多楼层多场景系统
  · 支持创建多个楼层图层，独立管理每层的建筑元素和人员单位
  · 楼层可自定义名称、上下排序、单独删除
  · 每个楼层内支持**多个场景**，场景间独立存储元素与枪线
    - 新建场景自动复制前一场景的全部内容
    - 场景支持自定义名称（桌面端双击/移动端长按重命名）
    - 桌面端 hover 显示 × 删除按钮，移动端长按弹出操作面板
    - 至少保留一个场景不可删除
  · 楼梯元素可实现跨楼层连接
  · 默认起始楼层：1F 一楼（含默认场景1）


二、建筑元素系统
  支持以下建筑元素的绘制与管理：

  ┌──────────┬──────────────────────────────┐
  │  元素类型  │           功能说明            │
  ├──────────┼──────────────────────────────┤
  │  🧱 墙体   │ 矩形墙体，支持旋转，遮挡视野   │
  │  🚪 门    │ 可旋转开合，4向铰链位置可选     │
  │          │ 开合角度 0°~360° 可调          │
  │          │ 颜色跟随元素自身属性            │
  │  🪟 窗户  │ 半透明窗户，不遮挡视野          │
  │  🪜 楼梯  │ 连接不同楼层的通道             │
  │  🚧 遮挡物 │ 矩形或圆形遮挡物，支持旋转     │
  └──────────┴──────────────────────────────┘

  所有建筑元素均支持：
  · 自由拖拽移动
  · 0°~360° 旋转
  · 宽高尺寸调整
  · 颜色自定义
  · 标签标注


三、人员单位系统
  支持三类人员单位的部署：

  ┌──────────┬──────────────────────────────┐
  │  单位类型  │           功能说明            │
  ├──────────┼──────────────────────────────┤
  │  👤 操作员 │ 我方 CQB 操作员              │
  │          │ 支持设置：名称、呼号、职位       │
  │          │ 可作为枪线起点                 │
  │  💀 敌人  │ 敌对目标                       │
  │          │ 实战模式下可隐藏               │
  │          │ 可作为枪线起点                 │
  │  🙋 人质  │ 需要救援的目标                 │
  │          │ 不可作为枪线起点               │
  └──────────┴──────────────────────────────┘

  全局设置：
  · 操作员大小统一调节（10px ~ 30px）


四、枪线系统
  用于标记射击方向和火力覆盖范围：

  · 桌面端操作：右键长按操作员/敌人 → 拖动 → 松开完成
  · 移动端操作：单指长按人物（350ms）→ 拖动 → 松开完成
  · 枪线显示为带箭头的方向线，颜色与来源人物一致
  · 支持选中后单独删除
  · 长按时移动超过 10px 自动取消，防止误触


五、画板工具
  在战术图上进行自由标注：

  ┌──────────┬──────────────────────────────┐
  │  工具     │           功能说明            │
  ├──────────┼──────────────────────────────┤
  │  ✏️ 画笔  │ 自由手绘路径                  │
  │  ▭ 矩形   │ 绘制矩形框                    │
  │  ⭕ 圆形   │ 绘制圆形                      │
  │  📏 直线  │ 绘制直线段                    │
  │  🗑️ 区域删除│ 框选区域擦除画板内容         │
  │  🧹 清空  │ 一键清空当前场景所有画板内容    │
  └──────────┴──────────────────────────────┘

  画笔参数：
  · 颜色自由选取
  · 线宽 1px ~ 20px 可调


六、视野遮挡系统（核心特色功能）
  模拟真实室内环境中的视线遮挡效果：

  · 操作员拥有扇形视野锥（默认 120°）
  · 视野距离可设为有限或无限
  · 建筑元素（墙、门、楼梯、障碍物）会遮挡视线
  · **旋转的建筑元素**按每个角度实时计算遮挡区域
  · 门的开合状态影响视野穿透，4向铰链均正确计算
  · 窗户不阻挡视野穿透
  · 可全局开关视野遮挡显示
  · 射线投射算法（Raycasting）实现精确遮挡计算


七、实战模式
  · 开启后敌方目标自动隐藏，模拟真实情报不确定场景
  · 顶部显示 "实战模式已启用" 提示横幅
  · 快捷键 M 一键切换


八、智能对齐系统
  · 绘制时自动吸附到网格交点和其他元素边缘
  · 对齐阈值 8px
  · **尺寸吸附**：新建建筑元素时自动匹配已有元素的宽高
  · 创建时不自动连接附近元素，需手动贴合才吸附
  · 可随时开关（快捷键 S）


九、画布缩放与平移（希沃白板模式）
  支持对画布进行自由缩放和平移：

  ┌──────────┬──────────────────────────────┐
  │   操作    │         功能说明              │
  ├──────────┼──────────────────────────────┤
  │ 滚轮滚动  │ 以鼠标位置为中心缩放           │
  │ 选择模式拖拽│ 平移整个画布               │
  │ 双指捏合  │ 缩放画布（移动端）             │
  │ 单指拖动  │ 平移画布（移动端）             │
  └──────────┴──────────────────────────────┘

  缩放范围：25% ~ 500%
  坐标转换：所有鼠标/触摸事件自动转换为画布坐标


十、撤销 / 重做
  · 最多保存 50 步历史记录
  · 完整记录每个楼层的所有场景数据
  · 桌面端：Ctrl+Z 撤销 / Ctrl+Y 重做
  · 移动端：底部圆形按钮 ↩️ 撤回 / ↪️ 取消撤回


十一、导出功能
  · 将当前画布导出为 PNG 图片
  · 快捷键 Ctrl+S 或点击导出按钮

布局结构：
  ┌─────────┬──────────────────────┬─────────────┐
  │         │                      │             │
  │ 左侧工具栏│      主画布区域       │ 右侧属性面板 │

左侧工具栏包含：
  · 楼层管理区 — 新建/切换/排序/删除楼层 + **场景Tab栏**
  · 操作工具区 — 选择（默认）、删除、实战模式开关
  · 建筑元素区 — 墙体、门、窗户、楼梯、遮挡物
  · 画板工具区 — 画笔、矩形、圆形、直线及参数
  · 人员单位区 — 操作员、敌人、人质
  · 视野系统区 — 视野遮挡开关
  · 视图控制区 — 清空、导出、网格切换

  楼层管理区详情：
  ┌─────────────────────────────────────┐
  │ 1F 一楼              ▲ ▼ ×          │ ← 楼层名称行
  │ [场景1][场景2][场景3] [+ ]          │ ← 场景Tab横排
  ├─────────────────────────────────────┤
  │ 2F 二楼              ▲ △ ×          │
  │ [场景1] [+ ]                        │
  └─────────────────────────────────────┘

右侧属性面板包含：
  · 选中元素的详细属性编辑（位置/尺寸/旋转/颜色等）
  · 门专属控制（开合角度、铰链位置 4 向选择）
  · 视野参数（FOV 角度、距离范围）
  · 人员信息（名称、呼号、职位）
  · 枪线管理
  · 图例说明
  · 快捷键参考
  · 全局设置

UI 设计风格：
  · SpaceX 航空航天美学（纯黑背景 + 光谱白文字）
  · 大写字母铭牌风格（字间距、全大写）
  · 半透明幽灵边框/表面（Ghost Surface）
  · 楼层管理：透明卡片 + hover显操作按钮 + 药丸形场景Tab
  · 响应式适配桌面端与移动端


================================================================================
                             移动端适配
================================================================================

切换方式：点击工具栏顶部的「📱 移动端」/「💻 桌面端」按钮

移动端布局变化：
  · 左侧工具栏变为顶部横向滚动条
  · 工具按钮紧凑排列，支持横向滑动浏览
  · 属性面板变为右侧滑出式侧边栏
  · 底部居中显示两个独立圆形操作按钮：
    - ↩️ 蓝色撤回按钮（50px 圆形）
    - ↪️ 绿色取消撤回按钮（50px 圆形）

移动端手势操作：
  ┌─────────────────────────────────────────┐
  │ 手势           │ 操作说明                │
  ├─────────────────────────────────────────┤
  │ 单指点击       │ 选择元素 / 开始绘制新元素 │
  │ 单指拖动       │ 移动已选中元素 / 平移画布 │
  │ 双指捏合       │ 缩放画布                 │
  │ 长按人物(350ms)│ 进入枪线模式 → 拖出枪线  │
  │ 长按场景Tab   │ 弹出操作面板（重命名/删除）│
  └─────────────────────────────────────────┘

  切换至移动端时会弹出操作提示浮窗（3秒后自动消失）。

================================================================================
                              使用流程示例
================================================================================

  典型的 CQB 战术规划流程：

  1. 创建楼层架构
     → 新建楼层（如 1F 一楼、2F 二楼），用楼梯连接各层

  2. 绘制建筑平面图
     → 用墙体勾勒房间轮廓
     → 放置门窗并调整开合角度
     → 添加障碍物（家具/掩体）

  3. 部署人员
     → 放置我方操作员，设置呼号和职位
     → 放置敌对目标和人质

  4. 分析视野
     → 查看每位操作员的视野覆盖范围
     → 通过旋转门/调整站位优化视野
     → 开启视野遮挡查看实际可见区域

  5. 多场景推演
     → 为同一楼层创建多个场景（如：初始部署/突入阶段/清房阶段）
     → 新建场景自动复制前一场景的全部元素
     → 在不同场景中独立调整人员部署和行动路线

  6. 规划行动路线
     → 用画板工具标注行进路线
     → 从操作员拖出枪线标记火力方向
     → 用矩形/圆形标记关键区域（突破点/集合点）

  7. 推演验证
     → 开启实战模式隐藏敌方，进行盲推演练
     → 在各场景间快速切换对比方案
     → 导出图片分享给团队成员
